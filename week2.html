Final Project Week 2 Journal

My final project is going to be an app album which includes an artwork viewer and an interactive story featuring Marat & Yarolslav, founders of Russian funk outfit ‘French Fries’. This is an extension of my Individual Design project where the characters Marat and Yaroslav were first created. The idea for this project came from research during the Individual Design module, which got me thinking about the concept of app as album.

Development:

I’ve started developing the visual style and concept for the interactive novel/short story, of which I’m pleased with so far. I’ve managed to place the basic artwork into an HTML file and with the use of Impress.js, I’ve made basic transitions from section to section, which would work well for an interactive graphic novel. I’d like to develop the graphic style further, using acetates which would be layered to create the overall image. I'd like to then animate each acetate (PNG file) in JavaScript.

I’ve identified a few JavaScript libraries that I could possibly use: Three.js, Pixi.js and Popmotion.js. So far Popmotion looks the easiest to implement. It includes a drag feature which I’m interested in exploring as I’d like the artwork in the artwork viewer to be able to be moved around by the user, thus creating their own version of the album artwork.
